# HERRAMIENTAS PARA AUTOMATIZAR LA OBRA

# Propósito del proyecto

Este proyecto nace con una idea :

Dotar al jefe de obra de herramientas gratis para automatizar tareas, gestionar datos, visualizar información y trabajar de forma más eficiente .

El objetivo es facilitar un conjunto de herramientas, guías y recursos que puedan crecer con el tiempo y servir como base para la digitalización real en obra.

# IA como asistente del jefe de obra

Como la IA puede ayudar al jefe de obra. Estado: investigación inicial.

# Herramientas open source para construcción

Guías prácticas de herramientas libres: LibreOffice, ProjectLibre, Inkscape, QGIS, Blender, Freecad, Trilium Notes, etc. Estado: primeras fichas en preparación.

# Visión general del ecosistema

El proyecto se organiza en varios módulos independientes, cada uno con su propio repositorio. Todos están pensados para ser accesibles, prácticos y orientados al día a día .

 # Streamlit 
 
Aplicaciones sencillas para diferentes tareas como gestionar mediciones, fotos, presupuestos, pdf, etc. Estado: primeras versiones en desarrollo.

# Python 

Scripts prácticos para automatizar tareas habituales: mediciones, informes, control de costes, procesado de datos. Estado: en construcción.

# Bases de datos -SQLite 

Modelo de datos ligero y portable para sustituir Excel y permitir automatización real. Estado: diseño inicial.

# Blender 

Scripts y herramientas para visualizar obra en 3D, medir volúmenes y generar modelos simples. Estado: planificado.

# Estado actual del proyecto

Este ecosistema está en fase inicial. Estoy construyendo los primeros módulos y aprendiendo GitHub al mismo tiempo.

El desarrollo será gradual, transparente y orientado a crear una base sólida que pueda crecer durante los próximos años.
